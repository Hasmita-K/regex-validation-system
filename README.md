# Regex Validation System

## 📌 Project Overview
This project is a simple web-based application that uses **Regular Expressions (Regex)** to validate user inputs such as email, phone number, username, and password.

## 🚀 Features
- Email validation using regex
- Phone number validation (Indian format supported)
- Password strength validation
- Username validation
- Real-time input feedback
- Dynamic error messages

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript (Regex)

## 🎯 Objective
To understand and implement regex for real-world input validation in web applications.

## 📸 Output
The application highlights:
- ✅ Valid inputs in green  
- ❌ Invalid inputs in red  

## 📚 Learning Outcomes
- Learned regex pattern creation and optimization
- Improved JavaScript DOM manipulation skills
- Understood importance of input validation
